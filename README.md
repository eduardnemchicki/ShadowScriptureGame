A gamejam project (with shadow theme) that i decided to bring to playable state. Mainly was created as a learning tool for me, to improve my Unity and C# development skills by building a complete game from start to finish, 
including gameplay programming, UI, save systems, input handling, and level logic.
models, textures, sounds and music are free assets from unity store and freesound. All scripts are made by me, by hand, AI was used for tips on unity libraries and inspector interactions and to add comments
to places i neglected to do so while crunching for gamejam.

Controls:
| Action                         | Default Key   |
| ------------------------------ | ------------- |
| Move candle when it is up      | W/A/S/D       |
| Interact with clickable objects| Mouse Buttons |
| Pick up / Put down candle      | Space         |
| Highlight interactable objects | Left Shift    |
| Pause Menu                     | Esc           |

Most controls can be rmapped from Options menu.

Notes:
* The game contains one complete level.
* Completing the puzzle spawns a set of keys that trigger the ending.
* The Level Select menu is currently a placeholder.
* On the first launch, audio sliders may default to zero. If the game is silent, adjust the Music and Sound sliders in the Options menu.
